# 🧮 formalization

This repository is the **official release-facing formalization surface** of the **Panta Rhei framework**.

It presents the public formalization lane of the Atlas.

## What lives here

- official formalization-facing documentation
- stable public references to the formal layer
- generated wiki mirrors for the formalization lane
- release-facing formalization notes

## What does not live here

- active day-to-day community development of the Lean code
- experimental implementation branches
- contributor-first issue/PR flow

For active formal development, use:
- 🧮 https://github.com/Panta-Rhei-Framework/taulib

## Why this repo exists

The formal layer needs a clean public release surface separate from the contributor-facing engineering workspace.
